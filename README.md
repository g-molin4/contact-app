# Contact App - Laravel CRUD

Contact management application developed in **Laravel 10**, allowing users to **create, list, edit, and delete contacts**.  

---

## Features

- List all contacts.
- Create new contacts.
- Edit existing contacts.
- Delete contacts.
- Field validation (name, email, phone).
- Success messages when creating, updating, or deleting contacts.

---

## Requirements

- PHP >= 8.1
- Composer
- MySQL or another compatible database
- Node.js and NPM (for assets, if using Laravel Mix or Vite)
- Web server (Apache, Nginx) or `php artisan serve` for development


Português
# Contact App - CRUD Laravel

Aplicação de gerenciamento de contatos desenvolvida em **Laravel 10**, permitindo **criar, listar, editar e excluir contatos**.  

---

## Funcionalidades

- Listar todos os contatos.
- Criar novos contatos.
- Editar contatos existentes.
- Excluir contatos.
- Validação de campos (nome, email, telefone).
- Mensagens de sucesso ao criar, atualizar ou excluir contatos.

---

## Requisitos

- PHP >= 8.1
- Composer
- MySQL ou outro banco compatível
- Node.js e NPM (para assets, caso use Laravel Mix ou Vite)
- Servidor web (Apache, Nginx) ou `php artisan serve` para desenvolvimento
